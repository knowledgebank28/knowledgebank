== Description ==

Create a fullscreen background slider for your site in a couple of minutes with multiple images.
You can choose between 2 slider transitions, fade and slide, also you can set the slide change speed.

== Installation ==

1. Upload `fs-slider` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= I get an extra vertical sidebar when using the plugin =

If you get an extra vertical sidebar when using the plugin you can check the "Check this if you are experiencing an extra vertical scrollbar on the frontend" setting which fixes most the problem in most cases.
